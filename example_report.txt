ABOUT RMM SOFTWARE:
-------------------
Remote Monitoring & Management (RMM) tools are commonly used by IT
Managed Service Providers (MSPs) to remotely manage and monitor client
systems. Legitimate uses include:
  - Patch management and software updates
  - Remote troubleshooting and support
  - System health monitoring
  - Security scanning

However, attackers increasingly abuse these legitimate tools to:
  - Establish persistent remote access to compromised systems
  - Bypass endpoint security (tools are trusted/signed)
  - Move laterally across networks
  - Exfiltrate data while appearing as normal IT traffic

If you did not authorize the installation of remote management software,
treat any detected tool as a potential security incident.
